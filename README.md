# 📊 HR Analytics Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **HR Analytics Dashboard** developed using **Microsoft Power BI**. The dashboard provides key insights into employee demographics, workforce distribution, promotion eligibility, salary analysis, and employee attrition. It is designed to help HR professionals monitor workforce trends and support data-driven decision-making.

---

## 🖼 Dashboard Preview

## 🎯 Project Objectives

- Analyze the company's workforce.
- Monitor employee attrition.
- Evaluate promotion eligibility.
- Compare gender distribution.
- Analyze employee levels and years of service.
- Understand employee distance from home.

## 💡 Business Insights

- The organization consists of **1,470 employees**.
- Employee attrition rate is **16.1%**.
- The average monthly salary is approximately **$6,503**.
- Most employees are in **Job Levels 1 and 2**.
- A relatively small percentage of employees are currently eligible for promotion.
- Most employees live close to their workplace.
